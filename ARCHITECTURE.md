# Architecture Documentation

## Frontend Architecture

### HTML
`index.html`
- Main application entry point
- Contains all page sections

### CSS

#### variables.css
Design system:
- Colors
- Typography
- Spacing
- Shadows
- Animation timings

#### layout.css
Layout management:
- Containers
- Sections
- Footer
- Hero layouts

#### components.css
Reusable UI components:
- Buttons
- Cards
- Navigation
- Forms
- Stepper

#### animations.css
Visual effects:
- Transitions
- Motion effects
- Reduced-motion support

#### responsive.css
Responsive behavior:
- Tablet layouts
- Mobile layouts
- Breakpoint fixes

### JavaScript

navigation.js
- Smooth scrolling
- Active navigation state
- Mobile menu

splitText.js
- Character splitting animations

decryptText.js
- Text scrambling effect

spark.js
- Background particle network

stepper.js
- Interactive workflow stepper

serviceCycle.js
- Service rotation logic

capabilities.js
- Capability section interactions

app.js
- Application bootstrap
- Contact form handling
- UI interactions
